# Playlist Music
Um site de playlist de música que tem as funcionalidades de pausar e dar play na música, ir para proxima ou voltar uma música e ver a duração da música.

## Linguagens utilizadas

<ul>
 <li>HTML</li>
 <li>CSS</li>
 <li>JavaScript</li>
</ul>
